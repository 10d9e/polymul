## Approach

Drop internal reductions in the final inverse DIT stage
The final inverse DIT stage's outputs feed only the psi^{-1}*N^{-1}
post-weight Shoup, which tolerates any value < 2^32. With the 27-bit
primes the s0/s1/s2 conditional reductions can therefore be dropped:
outputs grow to [0,8p) (~2^30, well under 2^32) instead of [0,4p).
Adds a dedicated r4_lazy_dit_l_final butterfly used only by that stage
(the untwiddled input is still reduced so the sums stay bounded).
